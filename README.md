### Quick Start

Install Milestone by running either of the following:

- Install NodeJS LTS from [NodeJs Official Page](https://nodejs.org/en/)
  (NOTE: Product only works with LTS version)

- Install Yarn CLI from [Yarn Official Page](https://yarnpkg.com/cli/install)

Clone the repository with the following command:

```bash
git clone https://github.com/Avirage-official/milestone-clubhouse.git
```

Run in terminal this command:

```bash
yarn install
```

Then run this command to start your local server

```bash
yarn dev
```

### Reporting Issues

We use GitHub Issues as the official bug tracker for Milestone. Here are
some advices for our users that want to report an issue:

1. Make sure that you are using the latest version of the Milestone Dashbaord.
   Check the CHANGELOG from your dashboard on our
   [CHANGE LOG File](https://github.com/Avirage-official/milestone-clubhouse/blob/main/CHANGELOG.md).
2. Providing us reproducible steps for the issue will shorten the time it takes
   for it to be fixed.
3. Some issues may be browser specific, so specifying in what browser you
   encountered the issue might help.
